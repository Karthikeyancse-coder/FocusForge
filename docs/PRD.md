# FocusForge AI - Product Requirements Document

## 1. Product Vision
FocusForge AI transforms productivity into an adventure. Instead of a standard todo list, the application utilizes a gamified character evolution system combined with an AI-driven planner to help students avoid missed deadlines, optimize their focus, and visualize personal growth.

## 2. Core Problem
Students suffer from poor task prioritization, ignored passive reminders, and a lack of visual growth which leads to burnout and abandoned productivity software.

## 3. Target Users
- Primary: College Students, Test Aspirants.
- Secondary: Self-Learners, Early-career Professionals.

## 4. Key Features (MVP)
- **Dashboard Overview**: Quick glance at focus hours, upcoming high-risk tasks, and character level progress.
- **AI Task Prioritization Engine**: Tasks are evaluated by a risk algorithm measuring deadline proximity against completion progress. 
- **Character Evolution System**: Users grow an avatar by completing focus sessions and tasks. Features XP progression.
- **Boss Battle System**: Major projects/deadlines are modeled as "boss monsters", with HP dropping as tasks are accomplished.
- **Theme System**: Full light/dark mode support with token-based semantic coloring that transitions seamlessly in real-time.

## 5. Non-Scope for MVP
- Multiple Character Classes
- Full 3D World (Guilds, multiplayer setups)
- Voice Assistant
- Outlook Integration
- Premium Monetized Features

## 6. Success Metrics
- Average session length per user.
- Daily active streaks.
- Reduction in missed deadlines among active users.
